# House Price Estimator with ML Model, FastAPI and HTML

## ML Model
Used Linear Regression to make model to predict prices based on grade, sqft above and sqft living. No libraries were used for ml only maths. For training purpose approx dataset of 20k was used and for testing around 1k

## FastAPI
Used fastapi to make api /predict route to predict the price and send response to each requests

## HTML
Used html with tailwind css and script tag to implement or make an interactive interface for this prediction model.
